Fractal trees with Canvas

This is a demonstration how you can create fractal trees using the HTML5 canvas element. If you have a browser that supports canvas then you can see a form and a generated tree below. If not, you see a picture of what you could create if you used a new browser. You can save the image of the tree by right-clicking them in Firefox or dragging them out of the window in other browsers.

Written by Chris Heilmann - @codepo8 - inspired to build it at a workshop by @seb_ly.
